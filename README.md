Whatthehive
===========

Simple photo sharing web app with a RESTful API, based on [PocketBase][0]

## Collections

- `tags`

![tags.png](img/tags.png)

- `photos`

![photos.png](img/photos.png)


## Custom routes

- HTML: `/:tag_id`
- JSON: `/:tag_id/json`
